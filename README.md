# Responsive Multi-Page Blog

A minimal, fully responsive multi-page blog website built using pure HTML5, CSS3, and Vanilla JavaScript.

## Features
- **Real-Time Search:** Instantly filter blog posts by title on the homepage.
- **Mobile Responsive:** Works perfectly on desktop, tablet, and mobile with a functional hamburger menu.
- **Modern Layout:** Utilizes CSS Grid and Flexbox for clean spacing.
- **No Dependencies:** Built with 100% pure native web technologies.

## File Structure
- `index.html` - Homepage featuring the blog feed and search bar.
- `blog1.html` through `blog4.html` - Individual post detail pages.
- `style.css` - Main stylesheet handling layout, typography, and animations.
- `script.js` - JavaScript handling the mobile menu toggle and search logic.

## How to Run
1. Download all the project files into the same folder.
2. Double-click `index.html` to open and run the website in any browser.
